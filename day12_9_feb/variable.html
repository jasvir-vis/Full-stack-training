Variables in JavaScript

In JavaScript, variables are containers for storing values. You can declare them using var, let, or const.

Differences

  • var - Function-scoped, allows redeclaration, can be reassigned.
    1. var a = 10; // declaration
      var a = 20; // redeclaration
      a = 30; // re assign
  • let – Block-scoped, cannot be redeclared in the same block, but can be reassigned.
    1. var a = 10; // declaration
      var a = 20; // redeclaration not allowed
      a = 30; // re assign
  • const – Block-scoped, cannot be redeclared or reassigned (value stays fixed).
    1. var a = 10; // declaration
      var a = 20; // redeclaration not allowed
      a = 20; // reassign not allowed
